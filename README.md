 📖 Overview
**DarkCracker** is a powerful terminal-based (TUI) security framework designed for offensive security and penetration testing. It features a modular structure including WiFi scanning, SSH/FTP brute-forcing, and integrated AI capabilities.

 🚀 Installation on Kali Linux
To get started with **DarkCracker**, follow these steps in your terminal:

1. Clone the repository

git clone [https://github.com/d4rksea-team/darkcracker.git](https://github.com/d4rksea-team/darkcracker.git)
cd darkcracker


2. Setup Environment
You can use the automated installation script (Recommended for Kali):

bash install.sh


Or manually install the requirements:

pip install -r requirements.txt


🛠️ Usage
To launch the main application interface:

python3 main.py
